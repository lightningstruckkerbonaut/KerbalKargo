Kerbal Kargo is my first mod, adding a bunch of shipping containers and fluid tanks for hauling large amounts of resources.
Right now it's in a beta state, but there is compatibility with EPL, and it should be compatible with TAC LS, Kerbalism and Snacks (B9 subtypes with resources from those mods only if you have them installed)
Since it's in a beta state, it's likely that lots of stuff is gonna change especially if people end up finding bugs or mistakes which there should hopefully be none of

Dependencies:
B9PartSwitch
ModuleManager
